# Atividade Supervisionada - Desenvolvimento Mobile

### Listagem de alguns domínios de softwares que possam ser utilizados em Goiânia.

## Domínios
### Aplicativo que avise quando houver um alagamento na sua rota
 Utilizando a API do Google Maps e uma qualquer de tempo 

### Software que calcula o IMC e retorna uma dieta do dia
 Utiliza alguma API de pratos saudáveis ou uma lista fixada 

### Tradutor para tupi-guarani
 Tradutor sem propósito 

## Aplicativos que se relacionam

### Software de alagamento
- O Waze é semelhante, pois avisa quando ocorrem coisas assim, porém, não é em forma de notificação,
assim se não estiver com o mesmo ativado ou não olhar toda a rota, acaba caindo da mesma forma.

### Software de IMC
- Existem várias balanças de IMC na internet, mas são apenas balanças, nada voltamo mais para 
o que o cliente precisa fazer para melhorar ou o que precisa comer.

### Software Tradutor
- O Google Translate faz tradução de diversas línguas, mas não de línguas mortas como a 
tupi-guarani ou latim.
